# Kaula Language Project

## 项目概述

Kaula语言的设计理念注重性能与灵活性的平衡，其核心特性如VO（Virtual On-site）系统和Spend/Call机制为资源管理和调用提供了高效解决方案。VO系统通过缓存数据和代码，减少重复计算和内存开销，特别适合处理频繁访问的资源；Spend/Call机制则支持动态组件管理，提升代码的模块化和可维护性。

此外，Kaula的三级优先级队列系统优化了任务调度，确保高优先级任务优先执行，适合需要实时响应的场景。标准库针对Windows平台进行了深度优化，提供了丰富的模块如并发、异步、I/O等，简化了复杂功能的实现。

开发团队目前专注于Windows平台的稳定性和性能优化，未来计划扩展到其他平台。Kaula的模块化设计使得编译器、运行时和标准库可以独立演进，便于社区贡献和功能扩展。

**注意：当前版本仅支持Windows操作系统。**

## 核心功能

### 1. 编译器（kaula-compiler）

Kaula编译器使用Go语言实现，基于递归下降解析器，包含以下核心组件：

- **词法分析器（Lexer）**：将源代码转换为标记流，支持关键字、标识符、字面量、运算符、分隔符和注释的识别
- **语法分析器（Parser）**：使用递归下降法构建抽象语法树（AST），支持各种语句和表达式的解析
- **语义分析器（Semantic）**：进行类型检查、作用域分析和语义验证，确保程序的正确性
- **代码生成器（Codegen）**：基于模板生成C语言代码，然后通过C编译器编译为可执行文件

**技术细节**：
- 词法分析器使用状态机实现，缓存输入长度以提高性能
- 语法分析器支持复杂的嵌套结构和运算符优先级
- 代码生成器使用Go的template包生成C代码，支持插件扩展
- 编译器前端采用模块化设计，便于扩展和维护

### 2. 运行时系统（src）

运行时系统使用C语言实现，针对Windows平台优化，提供以下核心功能：

- **高精度时间测量**：使用Windows的QueryPerformanceCounter()实现纳秒级时间测量，支持周期到时间单位的转换
- **高速内存分配器**：实现了基于内存池的快速分配器，减少内存碎片和分配开销
- **VO (Virtual On-site) 系统**：提供高效的数据和代码缓存机制，支持数据和代码的关联
- **Spend/Call 机制**：实现资源管理和调用，支持组件的动态添加和调用
- **三级优先级队列系统**：支持高、中、低三级优先级任务调度，优化任务执行顺序

**技术细节**：
- 内存分配器预分配256MB内存池，使用偏移量管理内存分配，避免频繁的系统调用
- VO系统使用缓存机制，支持最多2048个数据项和代码项的缓存
- 优先级队列系统支持批量添加和执行任务，提高任务处理效率
- 运行时系统使用内联函数和宏定义优化关键路径，减少函数调用开销
- 所有组件均针对Windows平台特性进行了优化，确保最佳性能

### 3. 标准库（std）

标准库使用C语言实现，针对Windows平台优化，提供了丰富的功能模块：

- **基础类型**：整数、浮点数、布尔值、字符串等基础数据类型的实现
- **容器**：数组、列表、字典等数据结构的实现
- **并发**：基于Windows线程API的并发原语和同步机制
- **异步**：基于Windows回调机制的异步操作支持
- **I/O**：文件操作和网络操作的封装
- **数学**：常用数学函数和常量的实现
- **内存**：内存管理和内存池的高级接口
- **对象系统**：面向对象编程的基础支持
- **字符串**：字符串处理和操作的高效实现
- **系统**：Windows系统调用的封装和平台相关功能
- **任务**：基于优先级队列的任务创建和管理
- **VO**：VO系统的高级接口

**技术细节**：
- 标准库采用模块化设计，每个模块独立实现，便于维护和扩展
- 所有模块均针对Windows平台特性进行了优化
- 提供了统一的错误处理机制，便于错误跟踪和处理
- 标准库使用Makefile和build.bat脚本构建，支持不同编译环境
- 内存模块提供了内存池实现，减少内存分配开销

   - 三级优先级队列系统

   - 类型检查和语义验证

3. **灵活性**：
   - 支持函数式编程
   - 支持面向对象编程
   - 模块化设计

4. **并发支持**：
   - 内置并发原语
   - 任务调度系统
   - 异步操作支持

## 开发环境要求

- **编译器开发**：Go 1.21.0或更高版本
- **运行时和标准库开发**：C编译器（推荐使用MSVC，因为项目针对Windows平台优化）
- **操作系统**：仅限Windows（当前版本仅支持Windows平台）
